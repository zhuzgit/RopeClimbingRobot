CPU: STC32G12K128
RemoteControl：HOTRC华航DS605 SBUS version
Power：24V
Moto：DM-J8009P-2EC 



操作：
1.按钮3是机器人使能，按下后，显示在上部、白色为机器人运动使能，显示在下部、红色，为运动禁止。
2.按钮4是清除错误，当电机超压、欠压、过电流、MOS过温、线圈过温、通讯丢失、过载等错误时，用来清理错误。
3.摇杆前推，机器人前进。
4.摇杆后拉，机器人后退。
5.放松摇杆，机器人停止运动。
6.当晃动摇杆到适宜位置后，按下定速按键，机器人实现定速巡航功能，定速标志显示为红色，再次按下定速按键，取消定速巡航功能，定速标志显示为白色。
7.回传电压显示的是当前机器人工作电压。
